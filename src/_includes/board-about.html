This is a message board app. You can create new messages {% include 'buttons/button-ref.svg', button: 'buttons/add-button.svg' %} and edit the existing ones, and make new connections between messages.

This app is not m‑ld, but it uses a m‑ld engine to store the data and share it.

This app doesn't use a database. In fact this site is just statically served HTML and Javascript. The message board data is not stored anywhere except here in the web browser.

From a board, if you navigate somewhere else and come back, you'll see that the whole message board is still here.

If you use the Share button or copy the browser URL, then open a new incognito window, or a different browser, or on a different computer, and navigate to that URL, you'll see the same message board.

Now, if you make some changes... you'll see the state of the boards staying synchronised.

This sharing of state is achieved using m‑ld. There is no code in the app itself to communicate the data between the two boards. Instead, the app just makes the changes it wants to the data, and m‑ld handles keeping the board data synchronised.

m‑ld is told by the app which boards should share the same data. In this app, it's done using the URL fragment.

For more information about m‑ld, head to the documentation or the Frequently Asked Questions page.

Happy Messaging!

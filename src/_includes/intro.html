m-ld is a revolutionary new way of thinking about app data.

With m-ld, data is just local state, which is shared to everywhere that it is being used—on mobiles, in browsers, in microservices, anywhere.

At the heart of m-ld is a decentralised protocol for distributing state among clones. Using m-ld, every app instance has read-write access to the shared state via its local clone, with zero latency. Changes to the state are propagated to all other app instances, so that they are all eventually consistent.

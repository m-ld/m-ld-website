--- tags: faq question: Is my data secure? date: 2020-01-02 ---

m-ld data can be secured in motion and at rest.

In order for a clone to obtain and maintain the current app state, it must have access to the message broker. This means that the security of the data is dependent on the security of the message broker. This will typically comprise authentication of the user and encryption of the data on the wire.

A clone may also allow the data to be encrypted when persisted to disk. Where this is supported it is specific to the clone distribution.
